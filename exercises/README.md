# HTML Basics Assignment

## Assignment Title
HTML Fundamentals – Exercises 1, 2, and 3

## Description
This project demonstrates basic HTML concepts including:
- Page structure and content elements
- Forms and user input
- Tables for organizing data

Each exercise is implemented in a separate HTML file.

## Files Included
- `exercise1.html` – Simple webpage with headings, lists, links, and images
- `exercise2.html` – HTML form with various input elements
- `exercise3.html` – HTML table with merged cells and caption

## Steps to Run the Project
1. Clone or download the repository.
2. Ensure the repository is public.
3. Open any `.html` file in a web browser.
4. No additional setup is required.
