# Assignment: Web Design - Blues Café

## Brief Description
This project is a simple multi-page website for a café named **Blues Café**.
It includes a **Home**, **About**, and **Contact** page with:

- A banner image
- Icon-based navigation bar
- Content sections with text and images
- A contact form
- Consistent layout across pages

## files included
- `about.html` – Simple webpage containing about info
- `contact.html` – webpage with html form and contact info
- `index.html` – home page containing introduction
